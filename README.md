**Description:**  
A **feedback system for tourist attractions** that allows visitors to submit feedback, upload images, and view analytics.  
**Key Features:**  
- Collect user feedback and ratings  
- Upload images for better context  
- Admin dashboard for analytics and reporting  
**Tech Stack:** Python, Flask, SQLite, HTML/CSS, JavaScript  
**Potential Improvements:** Add sentiment analysis and map-based visualization of feedback.  
**GitHub:** [Tourist Feedback App](https://github.com/mrjawadd/tourist-Feedback)
